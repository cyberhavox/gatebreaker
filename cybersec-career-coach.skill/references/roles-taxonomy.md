# Cybersecurity Roles Taxonomy: Complete Hierarchy
## Every Role, Every Tier, Every Specialized Track

---

## 🧬 Phase 0 — Bedrock (Pre-Tier 1)

| Role | Focus |
|---|---|
| IT Support / Helpdesk Technician | Keeps systems running — teaches how they break |
| Network Technician | Physical and logical network infrastructure |
| Systems Administrator | OS internals, patch management, user access |
| Information Technology Analyst | Gateway into security; operational understanding |

---

## 🟢 TIER 1 — Entry Level (0–2 years)

> **Mentor**: John Hubbard (SOC), Tanya Janca (AppSec), Grace Hopper (Engineering), Carl Linnaeus (GRC)

| Role | Focus |
|---|---|
| SOC Analyst L1 | Alert triage, ticket handling, basic IR |
| Cybersecurity Analyst (Associate) | General monitoring, compliance support |
| Endpoint Security Analyst | EDR monitoring, host isolation |
| IT Security Intern | Shadow senior analysts, tool exposure |

| Junior GRC Analyst | Policy review, basic audit evidence collection |
| IT Auditor (Junior) | Compliance checks, documentation |
| Security Awareness Trainer (Associate) | Phishing simulation, user education |

**Entry Certifications**: Security+, Google Cybersecurity Certificate, eJPT, CompTIA CySA+

## 🔬 SPECIALIZED TRACKS (Cross All Tiers)

### Offensive Security Track
```
Security Helpdesk
  → Junior Penetration Tester (Tier 1)
    → Penetration Tester / Ethical Hacker (Tier 2)
      → Senior Penetration Tester / Red Team Operator (Tier 3)
        → Red Team Lead / Manager (Tier 4)
          → Director of Offensive Security (Tier 5)
```
*Expert Mentors*: Kevin Mitnick, HD Moore, Mudge/Peiter Zatko, Vivek Ramachandran

### Forensics & eDiscovery Track
```
Digital Forensics Analyst (Tier 2)
  → DFIR Specialist Senior (Tier 3)
    → Forensics Lead (Tier 4)
      → Head of DFIR / Director of Forensics (Tier 5)
```
*Expert Mentors*: Edmond Locard (Locard's Exchange Principle), Eric Zimmermann, Lenny Zeltser

### Threat Intelligence Track
```
Threat Intelligence Analyst (Tier 2)
  → CTI Senior Analyst (Tier 3)
    → Lead Threat Intelligence Analyst (Tier 4)
      → Director of Threat Intelligence (Tier 5)
```
*Expert Mentors*: Roberto Rodriguez, Florian Roth, Katie Nickels, Alan Turing

### GRC / Compliance Track
```
Junior GRC Analyst / IT Auditor (Tier 1)
  → GRC Analyst (Tier 2)
    → Senior GRC / Compliance Manager (Tier 3)
      → Director of GRC (Tier 5) → DPO / CRO (Tier 6)
```
*Expert Mentors*: Dr. Eric Cole, Florence Nightingale (board reporting), Carl Linnaeus (taxonomy)

### OT/ICS Security Track
```
OT Security Support
  → OT/ICS Security Analyst
    → ICS Security Engineer
      → OT Security Architect
        → Global Head of OT/ICS Security
```
*Expert Mentors*: Ralph Langner (Stuxnet), Robert M. Lee (ICS threat intel)

### AI/ML Security Track
```
AI Security Analyst
  → AI Security Researcher
    → Senior AI Security Researcher
      → AI Security Architect
        → Head of AI Safety / Director of AI Security (Tier 5-6)
```
*Expert Mentors*: Ian Goodfellow (GANs/Adversarial ML), Aryabhata (algorithmic foundation)

### Bug Bounty / Independent Research Track
```
Bug Bounty Hunter (Part-time) → Bug Bounty Hunter (Full-time)
  → Top-Ranked Program Hunter
    → CVE/0-day Discoverer
      → Independent Security Researcher
        → Security Researcher at Major Tech Company
```
*Expert Mentors*: Dan Kaminsky, James Kettle (PortSwigger), Katie Moussouris

### Academia / Policy Track
```
Security Researcher (Academic)
  → PhD in Information Security
    → Professor / University Researcher
      → Government Advisor
        → National Policy Lead / Cybersecurity Law Expert
```
*Expert Mentors*: Bruce Schneier (policy writing), Richard Clarke (US Cyber Czar), Shoshana Zuboff
